# switch-1-nodemcu
Switch 1 relay on NodeMCU platform (Lua)
---
### Directory structure:
    \*.lc - lua bytecode file
    \*.lua - lua files
    \lua\* - project source files
    \web\* - web server files
    \web\lua\* - web server service source files
    \firmware\* - NodeMCU custom build
    \schematic\* - схема электрическая
